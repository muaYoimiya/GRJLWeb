个人博客系统

项目描述

这是一个使用React和Node.js开发的个人博客系统，支持用户注册登录、文章发布、评论、分类等功能。系统采用前后端分离架构，前端使用React框架构建，后端使用Node.js和Express提供API接口，数据库使用MongoDB。

技术栈

React Node.js Express MongoDB Redux Bootstrap
